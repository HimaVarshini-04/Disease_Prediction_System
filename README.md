# Disease Prediction System

## Overview

Developed a machine learning–based disease prediction system using user symptoms as input. Implemented multiple algorithms including Decision Tree, Random Forest. Multiple machine learning algorithms, including Decision Tree,Random Forest were implemented to predict diseases accurately. Designed an interactive and user-friendly GUI using Python Tkinter for symptom selection. Displayed predicted diseases along with recommended precautions andmedications.


This project demonstrates Machine Learning concepts, data preprocessing, model training, and web application development using Python and Flask.

## Workflow

1. User enters symptoms through the web interface.
2. Symptoms are processed and converted into model inputs.
3. The trained machine learning model predicts the disease.
4. The predicted disease is displayed to the user.
5. The system provides recommended precautions and medications related to the predicted disease to help users take appropriate initial measures.
